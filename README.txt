A5 is a Direct Messaging platform. You can use this program to send and receive messages from other users on the 
ICS32 DSP platform.

Usage: Once you open the application, you will be able to either create a new user(using New) or login to an existing
user(using Open) in the File dropdown menu. From there, you can begin messaging friends or anyone you wish after you
select a contact from the contact list or add a new contact.  

A screenshot of the GUI communication example is included in the folder.
